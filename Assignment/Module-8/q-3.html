Q-3What are the default Bootstrap text settings?

Ans.

  • Bootstrap uses a default font-size of 1rem (16px by default), and its line-height is 1.5.
  • In Bootstrap text alignment set by it's class

    • ".text-start" => Start aligned text on all viewport sizes

    ".text-center" => Start aligned text on all viewport sizes

    ".text-end" => Start aligned text on all viewport sizes

  • Bootstrap styles HTML headings (<h1> to <h6> ) with a bolder font-weight and a responsive font-size. ex(."fs-1" to "fs-6"),.fw-bold,.fst-italic",.fw-lighter".
  • In Bootstrap text decoration set by it's class (".text-decoration-none",".text-decoration-underline".)
  • In Bootstrap text text Transform set by it's class (."text-uppercase", ".text-lowercase", ".text-capitalize")
